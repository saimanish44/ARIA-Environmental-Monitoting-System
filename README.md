# ARIA - Autonomous Risk Identification & Alert System

## Overview

ARIA is an ESP32-based environmental monitoring and emergency communication system designed to detect hazardous environmental conditions and provide real-time alerts.

The system integrates multiple sensors, intelligent threat classification, Wi-Fi, Bluetooth, and Li-Fi communication to improve safety and emergency response.

## Features

- Temperature Monitoring
- Humidity Monitoring
- Air Quality Detection
- Smoke Detection
- Flame Detection
- OLED Display Interface
- Wi-Fi Dashboard
- Bluetooth Communication
- Li-Fi Communication
- Emergency Alerts
- Relay-Based Safety Control

## Hardware Components

- ESP32
- Arduino Uno
- DHT22
- MQ135
- MQ2
- Flame Sensor
- OLED Display
- Relay Module
- RGB LEDs
- DFPlayer Mini

## Applications

- Smart Buildings
- Industrial Safety
- Environmental Monitoring
- Emergency Alert Systems
